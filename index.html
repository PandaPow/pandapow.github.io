Cardobot is an automated card-sorter for trading card game cards, such as Magic The Gathering or Pokemon. A stack of unsorted cards is placed within the robot, and each card is identified using image processing software. Cards are then picked up with a vacuum, and placed in a container according to the sorting criteria. Cards can be sorted by colour or value.

Submarine

A small submarine that can either be manually controlled by a tethered remote or autonomously controlled with an autopilot program. The autonomous program uses data from an IMU, camera, and depth (pressure) sensor.
